Apotris

Apotris is a free, open-source, multiplatform block-stacking game with highly responsive controls, deep customization, and 14 game modes. It offers both casual and competitive play, including local multiplayer, and can be played on many platforms beyond its original Game Boy Advance design.

Techmino

Techmino is a fast-paced block-stacking game focused on competitive and high-skill gameplay. It features multiple game modes, smooth and responsive controls, and advanced mechanics designed for experienced players, while still offering options for practice and customization.
